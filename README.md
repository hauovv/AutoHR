### XLAutoMAIL - 엑셀 오토메일

> 🎯 **XL-AutoMAIL V.1.0.1**  
> Excel 기반 Outlook 자동 메일 발송 도구  
> 수신자별 메일 제목 · 본문 · 첨부파일을 자동 구성하고, 클릭 한 번으로 대량 발송까지 처리할 수 있습니다.
>  
> 🔧 주요 기능
> - 수신자별 맞춤 메일 템플릿 작성  
> - 첨부파일 · 이미지 자동 삽입
> - Outlook 서명 적용, 예약 발송 가능
> - 개별 발송 or 전체 발송 모두 지원
